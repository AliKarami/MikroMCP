# v1.3 — PPPoE & OpenVPN Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add PPPoE client management and OpenVPN client/server management — 6 new tools, bringing the total from 82 to 88.

**Architecture:** Two new tool files (`pppoe-tools.ts`, `ovpn-tools.ts`) following the exact same patterns as `dhcp-client-tools.ts` and `ip-service-tools.ts`. PPPoE and OpenVPN client tools use add/update/remove (update is new in v1.3 — it diffs fields and returns `no_change` when nothing changed). OpenVPN server is a singleton resource (always exists, fetched as `records[0]`); its tool uses enable/disable/set actions. Passwords are never logged. All write tools carry `destructiveHint: true`.

**Tech Stack:** TypeScript, Zod, RouterOS REST API (`interface/pppoe-client`, `interface/ovpn-client`, `interface/ovpn-server/server`), Vitest.

### Key design decisions to keep in mind

- **PPPoE `running` field** — boolean (`true`/`"true"`) indicates connected; map to `status` filter enum `connected`/`disconnected`/`all`.
- **Update action (new in v1.3)** — build a `diff` array from fields that differ; if nothing differs (and no password provided), return `action: "no_change"` without calling `update`. Password is write-only in RouterOS — it never comes back in GET, so if provided it's always written.
- **OpenVPN server singleton** — `GET interface/ovpn-server/server` returns `T[]`; use `records[0]`. If the array is empty the OpenVPN package is not installed — throw `NOT_FOUND`.
- **OpenVPN server `enabled` field** — uses `"yes"`/`"no"` (not `disabled: true/false`).
- **`manage_ovpn_server set` requires ≥1 field** — validate upfront; throw `VALIDATION` if all configurable fields are `undefined`.
- **Passwords never in `log.info`** — only log `name`/`interface`/`action`.

import { z } from "zod";
import type { ToolDefinition, ToolContext, ToolResult } from "./tool-definition.js";
import type { RouterOSRecord } from "../../types.js";
import { enrichError } from "../errors/error-enricher.js";
import { MikroMCPError, ErrorCategory } from "../errors/error-types.js";
import { createLogger } from "../../observability/logger.js";

const log = createLogger("pppoe-tools");

const listPppoeClientsInputSchema = z
  .object({
    routerId: z.string().describe("Target router identifier from the router registry"),
    interface: z.string().optional().describe("Filter by parent interface name (exact match)"),
    status: z
      .enum(["connected", "disconnected", "all"])
      .default("all")
      .describe("Filter by connection status: connected (running=true), disconnected (running=false), or all"),
    limit: z
      .number()
      .int()
      .min(1)
      .max(500)
      .default(100)
      .describe("Maximum number of PPPoE clients to return"),
    offset: z.number().int().min(0).default(0).describe("Offset for pagination"),
  })
  .strict();

const listPppoeClientsTool: ToolDefinition = {
  name: "list_pppoe_clients",
  title: "List PPPoE Clients",
  description:
    "List PPPoE client interfaces on a MikroTik router with optional filtering by parent interface and connection status. Supports pagination.",
  inputSchema: listPppoeClientsInputSchema,
  annotations: {
    readOnlyHint: true,
    destructiveHint: false,
    idempotentHint: true,
    openWorldHint: false,
  },
  async handler(params: Record<string, unknown>, context: ToolContext): Promise<ToolResult> {
    const parsed = listPppoeClientsInputSchema.parse(params);
    log.info({ routerId: context.routerId, interface: parsed.interface, status: parsed.status }, "Listing PPPoE clients");
    try {
      let clients = await context.routerClient.get<RouterOSRecord>("interface/pppoe-client", {
        limit: undefined,
        offset: undefined,
      });

      if (parsed.interface) {
        clients = clients.filter((c) => (c as Record<string, unknown>).interface === parsed.interface);
      }

      if (parsed.status !== "all") {
        clients = clients.filter((c) => {
          const rec = c as Record<string, unknown>;
          const isRunning = rec.running === true || rec.running === "true";
          return parsed.status === "connected" ? isRunning : !isRunning;
        });
      }

      const total = clients.length;
      const paginated = clients.slice(parsed.offset, parsed.offset + parsed.limit);
      const hasMore = parsed.offset + parsed.limit < total;

      return {
        content: `PPPoE clients on ${context.routerId}: ${total} total, showing ${paginated.length} (offset ${parsed.offset})`,
        structuredContent: {
          routerId: context.routerId,
          clients: paginated,
          total,
          hasMore,
          offset: parsed.offset,
          limit: parsed.limit,
        },
      };
    } catch (err) {
      throw enrichError(err, { routerId: context.routerId, tool: "list_pppoe_clients" });
    }
  },
};

const managePppoeClientInputSchema = z
  .object({
    routerId: z.string().describe("Target router identifier from the router registry"),
    action: z.enum(["add", "update", "remove"]).describe("Action to perform"),
    name: z.string().describe("PPPoE client interface name — idempotency key"),
    interface: z.string().optional().describe("Parent interface (e.g. ether1; required for add)"),
    user: z.string().optional().describe("PPPoE username (required for add)"),
    password: z.string().optional().describe("PPPoE password (write-only — never returned by RouterOS)"),
    serviceName: z.string().optional().describe("PPPoE service name filter (leave blank to match any service)"),
    addDefaultRoute: z
      .boolean()
      .optional()
      .describe("Install a default route via this PPPoE connection"),
    dialOnDemand: z
      .boolean()
      .optional()
      .describe("Connect only when traffic is present (dial on demand)"),
    dryRun: z.boolean().default(false).describe("Preview changes without applying"),
  })
  .strict();

const managePppoeClientTool: ToolDefinition = {
  name: "manage_pppoe_client",
  title: "Manage PPPoE Client",
  description:
    "Add, update, or remove a PPPoE client interface. Idempotent by name: add returns already_exists when the same name, interface, and user already exist. Update returns no_change when all specified fields already match. Password is always written when provided because RouterOS does not expose it in GET responses.",
  inputSchema: managePppoeClientInputSchema,
  annotations: {
    readOnlyHint: false,
    destructiveHint: true,
    idempotentHint: true,
    openWorldHint: false,
  },
  snapshotPaths: ["interface/pppoe-client"],
  async handler(params: Record<string, unknown>, context: ToolContext): Promise<ToolResult> {
    const parsed = managePppoeClientInputSchema.parse(params);
    log.info({ routerId: context.routerId, action: parsed.action, name: parsed.name }, "Managing PPPoE client");
    try {
      const allClients = await context.routerClient.get<RouterOSRecord>("interface/pppoe-client", {
        limit: undefined,
        offset: undefined,
      });
      const existing = (allClients as Record<string, string>[]).find((c) => c.name === parsed.name);

      if (parsed.action === "add") {
        if (!parsed.interface) {
          throw new MikroMCPError({
            category: ErrorCategory.VALIDATION,
            code: "PPPOE_CLIENT_INTERFACE_REQUIRED",
            message: "interface is required when adding a PPPoE client.",
            details: { name: parsed.name },
            recoverability: {
              retryable: false,
              suggestedAction: "Provide the parent interface name (e.g. ether1).",
            },
          });
        }
        if (!parsed.user) {
          throw new MikroMCPError({
            category: ErrorCategory.VALIDATION,
            code: "PPPOE_CLIENT_USER_REQUIRED",
            message: "user is required when adding a PPPoE client.",
            details: { name: parsed.name },
            recoverability: {
              retryable: false,
              suggestedAction: "Provide the PPPoE username.",
            },
          });
        }

        if (existing) {
          if (existing.interface === parsed.interface && existing.user === parsed.user) {
            return {
              content: `PPPoE client "${parsed.name}" already exists with the same interface and user. No changes made.`,
              structuredContent: { action: "already_exists", client: existing },
            };
          }
          throw new MikroMCPError({
            category: ErrorCategory.CONFLICT,
            code: "PPPOE_CLIENT_CONFLICT",
            message: `PPPoE client "${parsed.name}" exists with different configuration.`,
            details: {
              existing: { interface: existing.interface, user: existing.user },
              requested: { interface: parsed.interface, user: parsed.user },
            },
            recoverability: {
              retryable: false,
              suggestedAction: "Remove the existing client first or use a different name.",
              alternativeTools: ["manage_pppoe_client"],
            },
          });
        }

        if (parsed.dryRun) {
          const diff = [
            { property: "name", before: null, after: parsed.name },
            { property: "interface", before: null, after: parsed.interface },
            { property: "user", before: null, after: parsed.user },
            ...(parsed.serviceName ? [{ property: "service-name", before: null, after: parsed.serviceName }] : []),
            ...(parsed.addDefaultRoute !== undefined
              ? [{ property: "add-default-route", before: null, after: parsed.addDefaultRoute ? "yes" : "no" }]
              : []),
            ...(parsed.dialOnDemand !== undefined
              ? [{ property: "dial-on-demand", before: null, after: parsed.dialOnDemand ? "yes" : "no" }]
              : []),
          ];
          return {
            content: `Dry run: Would add PPPoE client "${parsed.name}".`,
            structuredContent: { action: "dry_run", diff },
          };
        }

        const body: Record<string, string> = {
          name: parsed.name,
          interface: parsed.interface,
          user: parsed.user,
        };
        if (parsed.password) body.password = parsed.password;
        if (parsed.serviceName) body["service-name"] = parsed.serviceName;
        if (parsed.addDefaultRoute !== undefined) body["add-default-route"] = parsed.addDefaultRoute ? "yes" : "no";
        if (parsed.dialOnDemand !== undefined) body["dial-on-demand"] = parsed.dialOnDemand ? "yes" : "no";

        const created = await context.routerClient.create("interface/pppoe-client", body);
        log.info({ name: parsed.name, id: created[".id"] }, "PPPoE client added");
        return {
          content: `Added PPPoE client "${parsed.name}".`,
          structuredContent: { action: "created", client: created },
        };
      }

      if (parsed.action === "update") {
        if (!existing) {
          throw new MikroMCPError({
            category: ErrorCategory.NOT_FOUND,
            code: "PPPOE_CLIENT_NOT_FOUND",
            message: `PPPoE client "${parsed.name}" not found.`,
            details: { name: parsed.name },
            recoverability: {
              retryable: false,
              suggestedAction: "Verify the client name with list_pppoe_clients.",
              alternativeTools: ["list_pppoe_clients"],
            },
          });
        }

        const updates: Record<string, string> = {};
        const diff: Array<{ property: string; before: unknown; after: unknown }> = [];

        if (parsed.interface !== undefined && existing.interface !== parsed.interface) {
          updates.interface = parsed.interface;
          diff.push({ property: "interface", before: existing.interface, after: parsed.interface });
        }
        if (parsed.user !== undefined && existing.user !== parsed.user) {
          updates.user = parsed.user;
          diff.push({ property: "user", before: existing.user, after: parsed.user });
        }
        if (parsed.serviceName !== undefined && existing["service-name"] !== parsed.serviceName) {
          updates["service-name"] = parsed.serviceName;
          diff.push({ property: "service-name", before: existing["service-name"], after: parsed.serviceName });
        }
        if (parsed.addDefaultRoute !== undefined) {
          const current = existing["add-default-route"] === "yes" || existing["add-default-route"] === true;
          if (current !== parsed.addDefaultRoute) {
            updates["add-default-route"] = parsed.addDefaultRoute ? "yes" : "no";
            diff.push({ property: "add-default-route", before: existing["add-default-route"], after: updates["add-default-route"] });
          }
        }
        if (parsed.dialOnDemand !== undefined) {
          const current = existing["dial-on-demand"] === "yes" || existing["dial-on-demand"] === true;
          if (current !== parsed.dialOnDemand) {
            updates["dial-on-demand"] = parsed.dialOnDemand ? "yes" : "no";
            diff.push({ property: "dial-on-demand", before: existing["dial-on-demand"], after: updates["dial-on-demand"] });
          }
        }
        // Password is write-only in RouterOS — always update if provided, never compare
        if (parsed.password !== undefined) {
          updates.password = parsed.password;
          diff.push({ property: "password", before: "***", after: "***" });
        }

        if (Object.keys(updates).length === 0) {
          return {
            content: `PPPoE client "${parsed.name}" is already up to date. No changes made.`,
            structuredContent: { action: "no_change", name: parsed.name, id: existing[".id"] },
          };
        }

        if (parsed.dryRun) {
          return {
            content: `Dry run: Would update PPPoE client "${parsed.name}".`,
            structuredContent: { action: "dry_run", diff },
          };
        }

        await context.routerClient.update("interface/pppoe-client", existing[".id"], updates);
        log.info({ name: parsed.name }, "PPPoE client updated");
        return {
          content: `PPPoE client "${parsed.name}" updated.`,
          structuredContent: { action: "updated", name: parsed.name, id: existing[".id"], diff },
        };
      }

      // remove
      if (!existing) {
        return {
          content: `PPPoE client "${parsed.name}" not found. Nothing to remove.`,
          structuredContent: { action: "not_found", name: parsed.name },
        };
      }
      if (parsed.dryRun) {
        return {
          content: `Dry run: Would remove PPPoE client "${parsed.name}".`,
          structuredContent: {
            action: "dry_run",
            diff: [{ property: "name", before: parsed.name, after: null }],
          },
        };
      }
      await context.routerClient.remove("interface/pppoe-client", existing[".id"]);
      log.info({ name: parsed.name }, "PPPoE client removed");
      return {
        content: `Removed PPPoE client "${parsed.name}".`,
        structuredContent: { action: "removed", name: parsed.name, id: existing[".id"] },
      };
    } catch (err) {
      if (err instanceof MikroMCPError) throw err;
      throw enrichError(err, { routerId: context.routerId, tool: "manage_pppoe_client" });
    }
  },
};

const manageOvpnClientTool: ToolDefinition = {
  name: "manage_ovpn_client",
  title: "Manage OpenVPN Client",
  description:
    "Add, update, or remove an OpenVPN client interface. Idempotent by name: add returns already_exists when the same name and connectTo already exist. Update returns no_change when all specified fields already match. Password is always written when provided because RouterOS does not expose it in GET responses.",
  inputSchema: manageOvpnClientInputSchema,
  annotations: {
    readOnlyHint: false,
    destructiveHint: true,
    idempotentHint: true,
    openWorldHint: false,
  },
  snapshotPaths: ["interface/ovpn-client"],
  async handler(params: Record<string, unknown>, context: ToolContext): Promise<ToolResult> {
    const parsed = manageOvpnClientInputSchema.parse(params);
    log.info({ routerId: context.routerId, action: parsed.action, name: parsed.name }, "Managing OpenVPN client");
    try {
      const allClients = await context.routerClient.get<RouterOSRecord>("interface/ovpn-client", {
        limit: undefined,
        offset: undefined,
      });
      const existing = (allClients as Record<string, string>[]).find((c) => c.name === parsed.name);

      if (parsed.action === "add") {
        if (!parsed.connectTo) {
          throw new MikroMCPError({
            category: ErrorCategory.VALIDATION,
            code: "OVPN_CLIENT_CONNECT_TO_REQUIRED",
            message: "connectTo is required when adding an OpenVPN client.",
            details: { name: parsed.name },
            recoverability: {
              retryable: false,
              suggestedAction: "Provide the remote server address or hostname.",
            },
          });
        }

        if (existing) {
          if (existing["connect-to"] === parsed.connectTo) {
            return {
              content: `OpenVPN client "${parsed.name}" already exists with the same remote endpoint. No changes made.`,
              structuredContent: { action: "already_exists", client: existing },
            };
          }
          throw new MikroMCPError({
            category: ErrorCategory.CONFLICT,
            code: "OVPN_CLIENT_CONFLICT",
            message: `OpenVPN client "${parsed.name}" exists with a different remote endpoint.`,
            details: {
              existing: { connectTo: existing["connect-to"] },
              requested: { connectTo: parsed.connectTo },
            },
            recoverability: {
              retryable: false,
              suggestedAction: "Remove the existing client first or use a different name.",
              alternativeTools: ["manage_ovpn_client"],
            },
          });
        }

        if (parsed.dryRun) {
          const diff = [
            { property: "name", before: null, after: parsed.name },
            { property: "connect-to", before: null, after: parsed.connectTo },
            ...(parsed.port !== undefined ? [{ property: "port", before: null, after: String(parsed.port) }] : []),
            ...(parsed.mode ? [{ property: "mode", before: null, after: parsed.mode }] : []),
            ...(parsed.protocol ? [{ property: "protocol", before: null, after: parsed.protocol }] : []),
            ...(parsed.certificate ? [{ property: "certificate", before: null, after: parsed.certificate }] : []),
            ...(parsed.user ? [{ property: "user", before: null, after: parsed.user }] : []),
          ];
          return {
            content: `Dry run: Would add OpenVPN client "${parsed.name}".`,
            structuredContent: { action: "dry_run", diff },
          };
        }

        const body: Record<string, string> = {
          name: parsed.name,
          "connect-to": parsed.connectTo,
        };
        if (parsed.port !== undefined) body.port = String(parsed.port);
        if (parsed.mode) body.mode = parsed.mode;
        if (parsed.protocol) body.protocol = parsed.protocol;
        if (parsed.certificate) body.certificate = parsed.certificate;
        if (parsed.user) body.user = parsed.user;
        if (parsed.password) body.password = parsed.password;

        const created = await context.routerClient.create("interface/ovpn-client", body);
        log.info({ name: parsed.name, id: created[".id"] }, "OpenVPN client added");
        return {
          content: `Added OpenVPN client "${parsed.name}".`,
          structuredContent: { action: "created", client: created },
        };
      }

      if (parsed.action === "update") {
        if (!existing) {
          throw new MikroMCPError({
            category: ErrorCategory.NOT_FOUND,
            code: "OVPN_CLIENT_NOT_FOUND",
            message: `OpenVPN client "${parsed.name}" not found.`,
            details: { name: parsed.name },
            recoverability: {
              retryable: false,
              suggestedAction: "Verify the client name with list_ovpn_clients.",
              alternativeTools: ["list_ovpn_clients"],
            },
          });
        }

        const updates: Record<string, string> = {};
        const diff: Array<{ property: string; before: unknown; after: unknown }> = [];

        if (parsed.connectTo !== undefined && existing["connect-to"] !== parsed.connectTo) {
          updates["connect-to"] = parsed.connectTo;
          diff.push({ property: "connect-to", before: existing["connect-to"], after: parsed.connectTo });
        }
        if (parsed.port !== undefined && existing.port !== String(parsed.port)) {
          updates.port = String(parsed.port);
          diff.push({ property: "port", before: existing.port, after: updates.port });
        }
        if (parsed.mode !== undefined && existing.mode !== parsed.mode) {
          updates.mode = parsed.mode;
          diff.push({ property: "mode", before: existing.mode, after: parsed.mode });
        }
        if (parsed.protocol !== undefined && existing.protocol !== parsed.protocol) {
          updates.protocol = parsed.protocol;
          diff.push({ property: "protocol", before: existing.protocol, after: parsed.protocol });
        }
        if (parsed.certificate !== undefined && existing.certificate !== parsed.certificate) {
          updates.certificate = parsed.certificate;
          diff.push({ property: "certificate", before: existing.certificate, after: parsed.certificate });
        }
        if (parsed.user !== undefined && existing.user !== parsed.user) {
          updates.user = parsed.user;
          diff.push({ property: "user", before: existing.user, after: parsed.user });
        }
        // Password is write-only in RouterOS — always update if provided, never compare
        if (parsed.password !== undefined) {
          updates.password = parsed.password;
          diff.push({ property: "password", before: "***", after: "***" });
        }

        if (Object.keys(updates).length === 0) {
          return {
            content: `OpenVPN client "${parsed.name}" is already up to date. No changes made.`,
            structuredContent: { action: "no_change", name: parsed.name, id: existing[".id"] },
          };
        }

        if (parsed.dryRun) {
          return {
            content: `Dry run: Would update OpenVPN client "${parsed.name}".`,
            structuredContent: { action: "dry_run", diff },
          };
        }

        await context.routerClient.update("interface/ovpn-client", existing[".id"], updates);
        log.info({ name: parsed.name }, "OpenVPN client updated");
        return {
          content: `OpenVPN client "${parsed.name}" updated.`,
          structuredContent: { action: "updated", name: parsed.name, id: existing[".id"], diff },
        };
      }

      // remove
      if (!existing) {
        return {
          content: `OpenVPN client "${parsed.name}" not found. Nothing to remove.`,
          structuredContent: { action: "not_found", name: parsed.name },
        };
      }
      if (parsed.dryRun) {
        return {
          content: `Dry run: Would remove OpenVPN client "${parsed.name}".`,
          structuredContent: {
            action: "dry_run",
            diff: [{ property: "name", before: parsed.name, after: null }],
          },
        };
      }
      await context.routerClient.remove("interface/ovpn-client", existing[".id"]);
      log.info({ name: parsed.name }, "OpenVPN client removed");
      return {
        content: `Removed OpenVPN client "${parsed.name}".`,
        structuredContent: { action: "removed", name: parsed.name, id: existing[".id"] },
      };
    } catch (err) {
      if (err instanceof MikroMCPError) throw err;
      throw enrichError(err, { routerId: context.routerId, tool: "manage_ovpn_client" });
    }
  },
};

Add, update, or remove a PPPoE client interface. Idempotent by name: add returns `already_exists` when the same name, interface, and user already exist. | `user` | string | — | PPPoE username (required for add) |
| `password` | string | — | PPPoE password (write-only) |
| `serviceName` | string | — | PPPoE service name filter |
| `addDefaultRoute` | boolean | — | Install default route via this connection |
| `dialOnDemand` | boolean | — | Connect only when traffic is present |
| `dryRun` | boolean | `false` | Preview without applying |

**Example prompt:** "Add a PPPoE client named pppoe-wan on ether1 with username myisp and password secret123"

## manage_ovpn_server · Write · Destructive

Enable or disable the OpenVPN server, or update its configuration. Enable/disable are idempotent — returns `no_change` when already in the desired state. The `set` action returns `no_change` when all specified fields already match.

| Parameter | Type | Default | Description |

| `action` | enum | required | `enable`, `disable`, or `set` |
